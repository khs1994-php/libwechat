A WeChat PHP Library

## Installation

To Use WeChat SDK, simply:

```bash
$ composer require khs1994/wechat
```

For latest commit version:

```bash
$ composer require khs1994/wechat @dev
```

## Usage

```php
<?php

require __DIR__.'/vendor/autoload.php';

```
